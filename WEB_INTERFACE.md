# Nexus Cloud Gaming - Web Interface

## 🎨 Дизайн и Интерфейс

Веб-интерфейс Nexus Cloud Gaming разработан с использованием современных технологий и лучших практик UX/UI дизайна.

### Технологический стек
- **HTML5** - Семантическая разметка
- **CSS3** - Современный дизайн с анимациями
- **JavaScript ES6+** - Интерактивность

## 📱 Страницы и Секции

- Любое устройство

### 4. Секция Игр

Каждая категория показывает примеры популярных игр.

## 🔧 JavaScript Функциональность

### 1. Smooth Scroll
```javascript
document.querySelectorAll('a[href^="#"]').forEach(anchor => {
    anchor.addEventListener('click', function (e) {
        e.preventDefault();
        document.querySelector(this.getAttribute('href'))
            .scrollIntoView({ behavior: 'smooth' });
    });
});
```

### 2. Intersection Observer
Анимация элементов при прокрутке:
```javascript
const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            entry.target.style.opacity = '1';
        }
    });
});
```

## 🚀 Запуск Веб-Сервера

### Python 3
```bash
cd web
python3 -m http.server 8080
```

### Node.js (http-server)
```bash
npm install -g http-server
cd web
http-server -p 8080
```

### PHP
```bash
cd web
php -S localhost:8080
```

### Nginx (продакшн)
```nginx
server {
    listen 80;
    server_name nexuscloudgaming.com;
    
    root /path/to/web;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

## 🎨 Кастомизация

### Изменение цветов
В `styles.css` изменить переменные:
```css
:root {
    --primary-color: #ваш-цвет;
    --secondary-color: #ваш-цвет;
}
```

### Добавление новых игр
В `index.html` добавить в соответствующую категорию:
```html
<span>Название игры</span>
```

### Изменение текста
Просто отредактировать содержимое в `index.html`.
