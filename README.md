## 📦 Установка и Сборка

### Требования
- C++17 компилятор (GCC, Clang, MSVC)
- CMake 3.16+
- OpenSSL и libcurl

### На Linux/Mac
```bash
# Клонируем репозиторий
git clone https://github.com/nokiavisual-star/Nexus-Cloud-gaming.git
cd Nexus-Cloud-gaming

# Устанавливаем зависимости
sudo apt-get install libssl-dev libcurl4-openssl-dev  # На Ubuntu
brew install openssl curl  # На macOS

# Создаем директорию для сборки
mkdir build && cd build

# Компилируем проект
cmake ..
make

# Запускаем
./NexusCloudGaming
```

### На Windows
```cmd
# Используем Visual Studio или mingw
cmake -G "Visual Studio 16 2019" ..
cmake --build . --config Release

# Запускаем exe файл
NexusCloudGaming.exe
```

## 📊 Основные Компоненты

### 1. **StreamingServer** (src/server/streaming_server.h/cpp)
- Обработка потоков видео/аудио
- SSL/TLS шифрование
- Управление соединениями
- Компрессия и оптимизация

### 2. **GameManager** (src/server/game_manager.h/cpp)
- Управление библиотекой игр (5000+)
- Поиск и фильтрация
- Запуск и остановка игр
- Отслеживание статуса

### 3. **AuthManager** (src/core/auth_manager.h/cpp)
- Регистрация пользователей
- Аутентификация
- Генерация токенов
- Безопасность данных

### 4. **CloudStorage** (src/core/cloud_storage.h/cpp)
- Сохранение прогресса
- Синхронизация между устройствами
- История сохранений
- Восстановление данных

### 5. **InputHandler** (src/client/input_handler.h/cpp)
- Обработка клавиатуры
- Поддержка геймпадов
- Сенсорный ввод
- Управление мышью

## 🎮 Примеры Использования

### Запуск игры
```cpp
auto game_manager = std::make_unique<GameManager>();
game_manager->initialize_game_library();
game_manager->start_game("cyberpunk2077", "user_123456");
```

### Сохранение прогресса
```cpp
auto cloud = std::make_unique<CloudStorage>();
cloud->save_game_progress("user_123456", "elden_ring", 
                          R"({"level":50})", "PC");
```

### Аутентификация
```cpp
auto auth = std::make_unique<AuthManager>();
auth->register_user("username", "email@example.com", "password");
auth->login_user("email@example.com", "password");
```
